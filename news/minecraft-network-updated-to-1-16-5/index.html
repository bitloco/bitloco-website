Minecraft network updated to 1.16.5

Published on 2021-01-16

This is just a quick post to inform you that all our servers are now running 1.16.5 (updated from 1.16.4)!

If you're on vanilla (unmodified client), we recommend you to update right away.

This update is backwards compatible with multiplayer 1.16.4, so if you depend on mods that haven't updated yet, you can keep connecting with 1.16.4, but we recommend updating as soon as possible to fix some possible crashes.

Have fun playing!
