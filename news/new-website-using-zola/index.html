New website using Zola

Published on 2023-06-28

We're now running a new website!

It's basically the old content, but with different technology!

We moved from WordPress to Zola, which means this is now a static website!

OK but what does that mean?

This means you're getting a faster website, no more Cloudflare and no cookies!

The website is now more secure and privacy friendlier for you! Yay!

But also, even though it was a lot of work to port the website, I no longer have to take care of server maintenance or updates.

As server, I now use GitHub pages, so they take care of the servers (for free!), and since this is a static website, I also don't have to install any WordPress updates, since there are none! This saves time.
