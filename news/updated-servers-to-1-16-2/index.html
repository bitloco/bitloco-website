Updated Servers to 1.16.2

Published on 2020-08-28

Today I have updated the vanilla server from 1.16.1 to 1.16.2 which is a minor update. And spigot server from 1.15.2 to 1.16.2 which is a major update that introduces the new nether. Both servers are now on version pair. I will likely delete the old nether on the spigot server for it to regenerate the new nether to take advantage of all the new content this version provides, but first I will check if there are any major user builds in the nether as I wouldn't want to delete user creations. But of course, even if I delete the old nether I'll make a backup first. Stay tuned for more news.
